# ImageOptim Pro

🖼️ Redimensionneur et compresseur d'images par lot

## Fonctionnalités

- 📏 Redimensionnement avec conservation des proportions
- 🗜️ Compression configurable (10-100%)
- 📦 Conversion de format (JPEG, PNG, WebP)
- 🎨 Interface moderne en mode dark
- ⚡ Traitement par lot rapide
